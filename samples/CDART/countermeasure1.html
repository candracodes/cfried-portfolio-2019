About PennDOT CDART

CDART - PennDOT's Crash Data Analysis and Retrieval Tool
Version 003.010.000

PennDOT's Bureau of Planning and Research (BPR) Geographic Information Division (GID)

This website is the product of a cooperative effort between PennDOT's Highway Safety and Traffic Operations Divisions (HSTOD) and BPR.

CDART is a web-based query tool with the ability to present crash data in several formats including maps, spreadsheets, reports, and data files. The application allows the user to write queries using CRS data from 1997 - Current Year, which has been extracted and loaded into the GIS database.
